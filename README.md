# Features

Requires PHP: 5.6

- Animate.css
- Aos
- Bootstrap
- Chosen
- Counterup
- Ico Font
- Jquery
- Jquery Mask Plugin
- Jquery Easing
- Materialize
- Multiple Select
- Owl Carousel
- select2
